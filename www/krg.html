This interface provides on-line access to KRG, an HPSG-based grammar of Korean.

  • You can input one sentence at a time, analyze it using KRG, visualize analysis results in various forms and generate from a semantic representation.
  • KRG has a vocabulary of around 30,000 words, but not so many proper names.

This demonstration uses facilities of the (a) Linguistic Knowledge Builder (LKB) grammar engineering platform, (b) high-efficiency PET System parser and (c) the [incr tsdb()] profiling environment.

The demonstrator uses CSS and (some) JavaScript. Please relay bug reports and suggestions for improvement back to the KRG director or the KRG maintainer.
